# 中国二级行政区交互地图示例

此示例演示了如何使用 ECharts 加载本地 GeoJSON 与 CSV 数据，
将中国所有地级行政区（即二级行政区）展示在一个暗色/赛博朋克风格的交互式地图上。

## 功能亮点

- 地图级别为全中国地级市，可显示 300+ 个区划。
- 从 `行政区划名称-工作表1.csv` 动态解析游客足迹（Skipper、Kowalski、Pirate、Rico）。
- 四个复选框用于切换不同游客的到访区域。
- 根据选择的游客数量通过颜色深浅（红色系）显示区域热力。
- 悬停显示具体访客列表。
- 石墨色背景、霓虹边缘、入场动画，营造未来/赛博朋克风格。

## 运行方法

1. 确保工作目录下包含以下三个文件：
   - `test.html` （主页面）
   - `中华人民共和国.geojson` （地级市边界数据）
   - `行政区划名称-工作表1.csv` （游客足迹数据）

2. 由于浏览器的 `fetch` 不能直接读取文件系统，需要通过 HTTP 服务提供文件。
   例如在项目目录运行：
   ```bash
   python3 -m http.server 8000
   ```
   然后在浏览器中打开 `http://localhost:8000/test.html`。
   （也可以使用 VS Code 的 Live Server、Node `serve` 等工具）

3. 页面加载后即可看到地图。点击复选框切换访客，地图会高亮对应行政区。

4. 在浏览器开发者工具的控制台中可查看未匹配到 GeoJSON 的 CSV 名称，用于排查拼写差异。

### 新增功能 & 视图

- 页面顶部增加了三个标签，可在三种模式之间切换：
  1. **地级市地图** – 展示所有地级行政区；直辖市和特别行政区以整体呈现，省界以醒目边界叠加。
  2. **省级地图** – 展示省/直辖市/特别行政区边界，边界线条更粗，颜色表示访客数量。
  3. **省份访问表** – 显示按省份汇总的访客到访情况，勾选符号表示某游客曾到访该省。

- 切换到“省级地图”时仍可使用复选框筛选访客。
- 省级数据来源于 `中华人民共和国省.geojson`，请确保该文件放在同目录下。
